# BitLend L1

Native Bitcoin Lending on Layer 1 — Built on OP_NET

## 🚀 Overview

BitLend L1 is a native Bitcoin lending protocol designed to unlock DeFi directly on Bitcoin Layer 1 using OP_NET.

Bitcoin holds over $2 trillion in value, yet most of it remains idle. BitLend L1 transforms Bitcoin from a passive store of value into an active financial layer by enabling on-chain lending and borrowing — without bridges or wrapped assets.

## 🔒 Core Features

- Native BTC deposits
- On-chain lending & borrowing
- Smart collateral locking
- Automated interest calculation
- Transparent liquidation logic
- Fully Layer 1 execution via OP_NET

## ⚙️ How It Works

1. Lenders deposit BTC into the protocol.
2. Borrowers lock BTC as collateral.
3. OP_NET smart logic calculates interest and manages risk.
4. If collateral ratio drops below threshold, liquidation is triggered automatically.

## 🌍 Why It Matters

Most Bitcoin DeFi today depends on wrapping BTC or using bridges.  
BitLend L1 eliminates that risk by building directly on Bitcoin Layer 1.

No bridges.  
No wrapping.  
Just pure Bitcoin DeFi.
